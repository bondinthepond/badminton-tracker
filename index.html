What is Live Sync?

With Live Sync enabled, all score updates, player changes, and match status will sync in real-time across phones, tablets, and laptops. Perfect for tournaments where multiple people need to see live updates!

How to get Supabase credentials:
  1. Create a free account at supabase.com
  2. Create a new project
  3. Run the SQL from supabase-schema.sql in the SQL Editor
  4. Go to Settings → API
  5. Copy the URL and anon/public key
